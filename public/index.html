About GOAT Finder

GOAT Finder is a simple mini-app built to help users view, learn, and manage information about their goats and advertise them to other goat breeders. Created by Hayden Duran, web developer and goat breeding extrodinare. (Like tiger king but with goats).

This project is built with Node.js, Express, and MongoDB, and deployed on Render. The goal is to demonstrate a full-stack workflow from backend APIs to a simple Bootstrap-powered frontend.
